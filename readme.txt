== Description ==

View all WordPress Dashicons on one page. Adds a menu entry to the Tools menu.

== Installation ==

**Through WordPress' Plugin Repository**

1. Goto [wordpress.org/plugins/kt-dashicons-viewer](http://wordpress.org/plugins/kt-dashicons-viewer) and download the zip
2. Goto your WordPress. You can upload a zip-archive via `/wp-admin/plugin-install.php?tab=upload`

**Manual installation**

1. Upload all files found inside the zip archive to /wp-content/plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
